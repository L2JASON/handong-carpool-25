# handong-carpool-25

## 소개
한동대 학생들의 카풀 매칭을 돕는 서비스입니다.

## 팀원
- 이준혁(팀장)
- 성하민
- 문하영

## 사용 기술
- Frontend: React
- Backend: Flask
- DB: Firebase Realtime DB
- 지도: Kakao Map API

## 폴더 구조
```
project-root/
├── frontend/        # React 앱
├── backend/         # Flask API 서버
├── docs/            # 보고서, 발표자료
├── README.md
```

## 시작하기

### 1. 프론트엔드 실행
```
cd frontend
npm install
npm start
```

### 2. 백엔드 실행
```
cd backend
pip install -r requirements.txt
python app.py
```

### 3. 문서
- docs 폴더에 보고서 및 발표자료가 있습니다.
